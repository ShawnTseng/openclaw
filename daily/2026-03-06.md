### 技術亮點

#### **1. Prompt Engineering 優於 Code Logic**
- 不需要寫 `TemplateMatchingService.cs` 進行關鍵字匹配
- 直接透過 system prompt 指示 AI「看到 [範本] 就 100% 照貼」
- 更輕量、更靈活、更易維護

#### **2. [SPLIT] 訊息拆分機制**
- 長回覆自動分段成多則 LINE 訊息泡泡
- 提升用戶體驗（不會收到超長單則訊息）
- 系統自動處理拆分，AI 只需標記 `[SPLIT]`

#### **3. 真人模式自動轉接**
- 擴充觸發關鍵字（2個→7個）
- 文件中說明真人模式運作方式
- 自動切換後 AI 不再回應，直到管理員手動切回

#### **4. Temperature=0 策略**
- 確保回答一致性
- 避免同一問題每次回答不同
- 提高客戶體驗穩定性

#### **5. Context Overflow 防護**
- 自動裁切過長對話歷史
- 防止 token limit 溢出
- 保持系統穩定性
