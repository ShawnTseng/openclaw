## Lessons Learned

### Technical
1. **Safari OAuth** - Use Chrome for Google OAuth
2. **OpenClaw config** - Use `openclaw config set`, not CLI `config.patch`
3. **AppleScript timeout** - Large exports â†’ rsync
4. **Google Drive sync locks** - Stage in `/tmp/` first
5. **SK Extension Method limits** - Direct instantiation bypasses version constraints
6. **Gemma System Prompt** - Merge into first User message (no System role)
7. **Azure VM quota** - New subscriptions may have 0 quota; use Consumption Plan to bypass
8. **Azure region limits** - Japan East doesn't support gpt-4o-mini Standard; use East US
9. **Consumption Plan trick** - `az functionapp create --consumption-plan-location` bypasses VM quota entirely
10. **Bicep vs CLI** - Bicep for repeatable infra; CLI for one-time setup (RBAC, secrets)
