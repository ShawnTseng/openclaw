# 專案命名紀錄與建議

## 1. 命名討論背景
- 專案原名：MrvShopAI，為 88MRV 潮流選貨店打造的 LINE 智慧客服系統。
- 因應多租戶、多客戶需求，需將命名平台化、通用化，避免綁定單一品牌。
- 希望名稱簡短、好記、易於台灣人發音，並兼具個人品牌色彩與服飾電商定位。

## 2. 命名方向與原則
- 避免使用冷僻、難發音的單字。
- 結合 AI、服飾、電商、夥伴、創新等意象。
- 可考慮個人品牌（如 Shawn、Tseng）或正面、親切、潮流感的單字。
- 名稱結構建議：「品牌字」+「ShopAI」。

## 4. 最終命名建議

**Buddy ShopAI**

- 簡單、好記、親切，容易讓人產生信任感。
- 適合服飾電商智慧夥伴的品牌形象。
- 品牌定位：「你的電商服飾智慧好夥伴」

## 5. 願景與品牌精神
- 讓每個服飾品牌都能輕鬆擁有專屬、智慧、潮流的 AI 夥伴。
- 推動服飾電商產業的創新突破與顧客關係升級。
